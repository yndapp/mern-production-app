
# MERN Stack Application

Dummy MERN Stack App


## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express

**Database:** MongoDB



## Installation

Install my-project with npm

```bash
  git clone https://github.com/yndapp/mern-production-app.git
  cd mern-production-app
```
Start Server
```bash
  cd server
  npm install
  npm run start
  OR
  npm run dev
```
Start Client
```bash
  cd client
  npm install
  npm run start
  OR
  npm run dev
```
